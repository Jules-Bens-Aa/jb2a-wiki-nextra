## H2 Callouts

Callouts don't work when using the following documentation code.

```mdx {1}
import { Callout } from 'nextra/components'
 
<Callout type="info" emoji="ℹ️">
  Today is Friday.
</Callout>
```
To resolve this issue, follow these steps :



### Step 1
 Close the server and run the following command
``` filename="command" copy
pnpm install nextra nextra-theme-docs
```
### Step 2
The installation complete, modify the code for callouts as follow

```mdx {1} copy
import { Callout } from 'nextra-theme-docs'
 
<Callout type="info" emoji="ℹ️">
  Today is Friday.
</Callout>
```
### Step 3
Copy the code, paste the first line at the top of your mdx and the reste where you want your callout to display.
